A New Start

March 20, 2020
Misc Personal

Update: I finally decided to join the fediverse! It’s been fun so far!

Hi, Koakuma here~

Okay so… this blog started out as a publicly viewable version of my computer- related notes and experiances, especially with less common platforms like PowerPC. As time goes on, I feel like I should write about something better than just random, unorganized stuff.

But there’s one problem: I don’t know what to write about yet. Increasingly, I feel like I don’t have anything interesting to write about, either about me, my thoughts, or my computing experiences (which, to be honest, mostly consists of just trying to install a random OS on my Macs and see if it can be made comfortable enough to be my daily driver).

(At this point, I’m starting to wonder if a microblog like Twitter would be a better fit for me, since I can just post random photos/screenshots1 and do a stream-of-consciousness type posts, but I can’t handle the social media aspect of it very well, since it tends to stress me out.)

In the end… I don’t know. I think I’ll see if there’s anything I can write about. For the time being, I’ll just write about random stuff like usual. Maybe, in the future, when I do have something serious to write, I’ll do it.
